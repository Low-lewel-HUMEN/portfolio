# Features
Features main.c | main.exe:
* Create or open file `logger.log`

Features debugger-main.c | debugger-main.exe:
* Create or open file `logger.log`

# Technical details
info:
* main.c was created with target to create simple code for created `logger.log`,
* but I also decided to create debugger-main.c from main.c app through debugger (x64dbg)
* debugger-main.c = (type `DEBUGGER_ANALYS`)main.exe from main.c

# To uses, need download..
* Need download `git`
* Need download `gcc (C23+)`

# To uses, must be..
* `GCC` must be added to PATH
* `GIT` must be added to PATH

# How build this app?
* Open `cmd` or `PowerShell`
* Write command:
```
git clone https://github.com/Low-lewel-HUMEN/portfolio main
```
* Open `build-main.bat` or `build-debugger-main.bat` from folder `"main \ other \ dinamic-analys \ x64dbg \ only-winAPI-apps \ work-26.03.2026"`

# How start using this app?
* Open `main.exe` or `debugger-main.exe`

**Warning**: only windows x64.
